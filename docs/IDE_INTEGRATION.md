# IDE Integration Guide

## PyCharm Extension

The Heady PyCharm extension provides seamless integration with Heady systems. To install:

1. **Manual Build**:
   - Open the project in PyCharm: `extensions/pycharm`
   - Configure JDK: Use `C:\Users\erich\Java\jdk-21.0.2+13\`
   - Build via: Build > Build Project
   - Find JAR in `build/libs/heady-pycharm-1.0.0.jar`
2. **Direct Download**: [pycharm-extension.zip](file:///c:/Users/erich/Heady/distribution/pycharm-extension.zip)
3. Copy the generated JAR file to PyCharm's plugins directory
4. Restart PyCharm

### Features:
- Connect to Heady API
- Synchronize settings across devices
- Access Heady tools directly from the IDE
