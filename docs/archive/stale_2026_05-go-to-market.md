# Go-to-Market Plan: ASAP Execution

## ICP Sequencing

1. **Solo devs + indie builders** (fast adoption, feedback velocity)
2. **Small teams shipping products** (repeatable workflows)
3. **Platform/DevOps orgs** (governance + observability budgets)

## GTM Motions

### Motion A: Product-led (HeadyAI-IDE)

- Guided onboarding inside IDE
- 3 flagship demos: arena merge, consensus review, build audit playback
- **Metric:** time-to-first-successful-task

### Motion B: Ecosystem-led (HeadyMCP)

- "Safe MCP checklist" publication
- Curated connector packs (repos+CI, observability, cloud deploy)
- **Metric:** integration time reduction

### Motion C: Operator-led (HeadyCheck)

- "Tool overload" benchmark worksheet
- Consolidation case studies
- **Metric:** MTTR delta + tools eliminated

## ASAP Plan

| Phase | Focus | Pass Criteria |
|-------|-------|---------------|
| Foundation | Message house, 3 landing pages, onboarding docs | Pages live, <10min onboarding |
| Proof | 5-10 design partners, trust KPI measurement | Data shows fix-loop reduction |
| Scale | Weekly content, connector pack release | 50+ active users |
| Convert | Team plans, security posture doc, paid pilots | First revenue |

## KPIs

- **Activation:** time-to-first-success
- **Retention:** weekly active builders
- **Expansion:** surfaces used per account
- **Trust:** rollback rate + fix-loop time
- **Ops:** MTTR delta
