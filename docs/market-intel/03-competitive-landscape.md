## Battlecards (Starter)

### vs Copilot-first

- **They say:** "Fastest path to more code output"
- **We say:** "Fastest path to *trusted* output"
- **Must prove:** fix-loop reduction, consensus confidence, audit trails

### vs Observability Vendors

- **They say:** "Unified full-stack observability"
- **We say:** "Unified observability with intelligence-driven remediation"
- **Must prove:** faster MTTR, fewer tools required

### vs MCP Ecosystem

- **They say:** "Connect everything quickly"
- **We say:** "Connect everything safely"
- **Must prove:** approvals, allowlists, signed tool calls

## We Win When

- Buyer worries about correctness/security/auditability
- Tool sprawl is painful
- Scaling agentic workflows

## We Lose When

- They only want autocomplete
- They refuse governance overhead
- No security owner exists
