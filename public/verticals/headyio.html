FAQ

Frequently Asked Questions

Does the SDK require any external dependencies?

No. The Heady Hive SDK has zero dependencies ‚Äî it uses only Node.js built-in modules (http, https, events). This keeps it lightweight and secure.

What Node.js version is required?

Node.js 18 or higher. The SDK uses modern JavaScript features like optional chaining and async/await.

Can I connect to multiple Heady instances?

Yes. Create multiple HeadyClient instances with different URLs. Each maintains its own connection and auth state.

Do I need an API key?

For local development (localhost:3301), no key is needed. For production (headyme.com), use HEADY_API_KEY from your .env file.

What are the 155 auto-success tasks?

The auto-success engine continuously runs 155 background tasks across 10 categories: learning, optimization, integration, monitoring, maintenance, discovery, verification, creative, deep-intel, and hive-integration. They cycle every 16.18 seconds (œÜ-aligned).
