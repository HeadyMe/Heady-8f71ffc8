Connect to HeadyMCP

Get up and running in under 5 minutes. Works with Windsurf, VS Code, Claude Desktop, and any MCP-compatible client.
1

Sign Up for Beta Access

Head to the signup page and request beta access. You'll receive an API key immediately. An admin will authorize your account for full access.

Tip: Save your API key securely. You'll need it to authenticate all requests.
2

Configure Your MCP Client

Add HeadyMCP to your MCP client configuration. Here's the config for Windsurf / VS Code:

{
  "mcpServers": {
    "heady": {
      "url": "https://heady-manager-headysystems.onrender.com",
      "transport": "sse",
      "headers": {
        "X-API-Key": "YOUR_API_KEY_HERE"
      }
    }
  }
}

For Claude Desktop, add to ~/.claude/claude_desktop_config.json:

{
  "mcpServers": {
    "heady": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/client", 
               "--sse", "https://heady-manager-headysystems.onrender.com"],
      "env": {
        "HEADY_API_KEY": "YOUR_API_KEY_HERE"
      }
    }
  }
}
3

Verify Connection

Test that your connection is working by hitting the health endpoint:

curl -H "X-API-Key: YOUR_API_KEY" \
  https://heady-manager-headysystems.onrender.com/api/health

You should see:

{"ok":true,"service":"heady-manager","version":"2.0.0",...}

Example: Run the Full Pipeline

Trigger a full 5-stage pipeline run:

curl -X POST \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  https://heady-manager-headysystems.onrender.com/api/pipeline/run

Then monitor progress:

curl -H "X-API-Key: YOUR_API_KEY" \
  https://heady-manager-headysystems.onrender.com/api/pipeline/state

7

Billing & Usage

Your usage is tracked per API call. View your current usage by checking the admin dashboard or via API:

curl -H "X-API-Key: YOUR_API_KEY" \
  https://heady-manager-headysystems.onrender.com/api/billing/usage

Upgrade your plan anytime through the billing portal:

curl -X POST \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"plan":"starter"}' \
  https://heady-manager-headysystems.onrender.com/api/billing/checkout
