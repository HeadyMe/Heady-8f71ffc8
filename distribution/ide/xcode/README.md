# Heady Dev Companion for Xcode

Swift Source Editor Extension for Xcode 15+.

## Status: Planned

## Planned Features
- Source Editor Extension commands (Explain, Refactor, Tests, Docs, Fix)
- Xcode sidebar integration
- Agent mode for multi-file changes
- Voice input via macOS Speech framework

## Tech Stack
- Swift / SwiftUI
- XcodeKit (Source Editor Extension)
- URLSession for heady-api communication
